# Desafio Vai Car

Login page and user crud.

## Features

- React 16
- Webpack 4
- Babel
- Redux
- AWS (Cognito)

## Installation

- `git clone https://github.com/Buul/desafio-vai-car.git`
- cd desafio-vai-car
- yarn

### run

yarn start
